## Project Setup

### Prerequisites

- Node.js
- npm or yarn

### Getting Started

1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install` or `yarn install`
3. Start the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) in your browser
